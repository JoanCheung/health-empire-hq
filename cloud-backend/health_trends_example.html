API 端点信息

此示例展示了如何使用健康趋势 API 端点获取和展示数据：

获取健康趋势:
GET /api/v1/users/{user_id}/health-trends?time_range=30d&assessment_type=comprehensive
获取健康记录列表:
GET /api/v1/users/{user_id}/health-records?skip=0&limit=20
创建健康记录:
POST /api/v1/users/{user_id}/health-records
获取健康统计:
GET /api/v1/users/{user_id}/health-summary?time_range=30d
